## 📊 CREAR TU CUENTA ADMIN (después de que funcione):

1. Ve a: **http://localhost:3000/register**
2. Regístrate con tu email
3. Ve a Supabase Dashboard → Authentication → Users
4. Encuentra tu usuario
5. Ve a Table Editor → user_profiles
6. Cambia el campo `role` de `'client'` a `'admin'`
7. Ahora podés entrar al admin panel
